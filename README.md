# Student Performance Dashboard

## Project Overview
This project analyzes the performance of 30 students using Python.

The analysis is based on:
- Subject marks (Math, Science, English)
- Study hours
- Attendance percentage
- Class-wise performance

Libraries used:
- Pandas
- NumPy
- Matplotlib

--------------------------------------

## Dataset Columns
- Student_ID
- Class
- Gender
- Math
- Science
- English
- Study_Hours
- Attendance_%
- Average (calculated)

--------------------------------------

## Analysis Done
- Subject-wise average
- Lowest performing subject
- Study hours vs marks correlation
- Attendance vs performance correlation
- Class-wise performance
- Top 5 students
- Bar chart and Scatter plot visualization

--------------------------------------

## How to Run

Step 1:
pip install -r requirements.txt

Step 2:
python analysis.py

--------------------------------------

## Learning Outcome
- Data generation using NumPy
- Data analysis using Pandas
- Data visualization using Matplotlib
